# Dawn Adventurer

Source: 3D Anime Female Adventurer Character Model by Dawn to Dusk Games.

License: Creative Commons Zero v1.0 Universal (CC0), as stated on the asset page.

Imported files in this folder are converted from the original FBX package into GLB for Godot runtime use. Keep project references relative, for example `res://assets/characters/dawn_adventurer/...`.

Current fighting mappings:
- Idle: `one_handed_sword_idle.glb`
- Run: `one_handed_sword_run.glb`
- Punch/Kick/Heavy attack placeholders: `one_handed_sword_attack_01/02/03.glb`
- Knockdown/Get up source clips are included for future wiring.

Note: this pack does not include a dedicated jump animation. The current roster entry uses an attack clip as a temporary airborne visual until a better character-specific jump clip is added.
